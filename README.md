# سيرفر محمد عمر موتورز (نسخة تجريبية)

كود سيرفر بسيط (Node.js + Express) بيدير السيارات والحجوزات والحسابات
بشكل حقيقي ودائم، بدل الاعتماد على تخزين مؤقت.

## ⚠️ مهم
الكود ده **لازم تشغّله إنت بنفسك** — مش هيشتغل لوحده. ده مش موقع مرفوع
على الإنترنت، ده كود خام على جهازك تشغّله محليًا أو ترفعه على استضافة
زي Render أو Railway أو أي VPS.

## خطوات التشغيل محليًا
1. نزّل ونصّب Node.js من https://nodejs.org (اختار نسخة LTS)
2. افتح تيرمنال/CMD جوه مجلد `backend-starter`
3. اكتب: `npm install`
4. اكتب: `npm start`
5. السيرفر هيشتغل على: `http://localhost:3000`

## البيانات
البيانات بتتخزن في ملف `db.json` بيتعمل تلقائيًا أول ما تشغّل السيرفر.
ده كافي للبداية، ولو المشروع كبر تقدر تستبدله بقاعدة بيانات حقيقية
(PostgreSQL أو MongoDB) بدون ما تغيّر شكل الـ API.

## الـ API المتاح
- `GET /api/cars` — كل السيارات
- `POST /api/cars` — إضافة سيارة
- `PUT /api/cars/:id` — تعديل سيارة
- `DELETE /api/cars/:id` — حذف سيارة
- `GET /api/bookings?phone=01xxxxxxxxx` — حجوزات عميل معيّن
- `POST /api/bookings` — إنشاء حجز جديد
- `PUT /api/bookings/:id/status` — تغيير حالة الحجز
- `POST /api/accounts` — إنشاء/تحديث حساب عميل
- `GET /api/accounts/:phone` — بيانات حساب عميل

## الخطوة الجاية
عشان تربط ده بملفات الموقع (`mohamed-omar-motors.html`, `admin.html`)،
هتحتاج تستبدل استدعاءات `window.storage` بطلبات `fetch()` لنفس الروابط
دي، ورفع السيرفر على استضافة حقيقية (مش جهازك الشخصي) عشان يفضل شغال
باستمرار. لو حابب، ممكن نعمل الخطوة دي مع بعض لما يكون عندك استضافة
جاهزة.
